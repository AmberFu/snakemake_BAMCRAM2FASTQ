# snakemake_BAMCRAM2FASTQ
BAM or CRAM file to pair end FASTQ

> Docker Image: `snakemake/snakemake:stable`
> 
> Tools: samtools
> 
> INPUT file format: BAM or CRAM
> 
> OUTPUT file format: pair-end FASTQ
